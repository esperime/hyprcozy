# 📥 Installation
```
git clone https://github.com/ki-bun/dotfiles-hyprland.git
cd dotfiles-hyprland
./install.sh
```

# 🛠️ Usage
`SUPER + RETURN` - Launch Terminal emulator

`SUPER + D` - Launch Fuzzel

`SUPER + E` - Open file manager

`SUPER + Q` - Quit / Exit

`SUPER + L` - Launch logout menu

`PRINT` - Capture a window

`SHIFT + PRINT` - Capture fullscreen

`SUPER + PRINT` - Capture a region
